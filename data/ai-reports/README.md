# AI 리포트 정적 파일

이 폴더에 보고서 파일을 넣으면 Claude 재생성 대신 해당 내용이 그대로 표시됩니다.

## 파일명 규칙

`{year}-{month}-{yearType}-{mode}.txt`

예:
- `2026-2-actual-ytd.txt` — 2026년 2월 실적, YTD
- `2026-2-actual-monthly.txt` — 2026년 2월 실적, 당월
- `2026-12-plan-ytd.txt` — 2026년 예산, YTD

## 사용 방법

1. 로컬에서 AI 리포트 모달 열기 → 재생성으로 원하는 보고서 생성
2. **정적 파일로 저장** 버튼 클릭 → `data/ai-reports/`에 바로 저장 (로컬 전용)
3. git add & commit & push

파일이 있으면 해당 조건에서 항상 이 파일 내용이 사용됩니다. 새로 생성하려면 파일을 삭제 후 재생성 버튼을 누르세요.
